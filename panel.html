From Pencil to Heart

Pencil portrait art that captures emotion and quiet stories in every face.

About this work

This project combines my pencil portrait artworks with a simple web gallery. Visitors can view my drawings, imagine ordering prints, and see how a small personal brand can start from just a pencil and a page.

Technologies used

  • HTML for structure
  • CSS for layout, gradients, and responsive design
  • JavaScript for image modal pop‑ups (on the main site)

What I focused on

  • Matching the calm, soft feeling of pencil art with the web design.
  • Making sure the layout works on both phones and larger screens.
  • Letting viewers see the artwork clearly, without too many distractions.

Next steps

In the future, I want to add a real order form, multi‑language support, and maybe a simple system to manage
